# test-mini-site

This is a simple restaurant menu website with multi-language support (English, Russian, Chinese).

## Features
- Menu with 15 items
- Language switcher for English, Russian, and Chinese
- Responsive design

## How to Run
1. Open a terminal in the project directory.
2. Run `python3 -m http.server 8000`
3. Open your browser and go to `http://localhost:8000`

## For QR Code Access
To make the site accessible via QR code:
1. Deploy the site to a web hosting service (e.g., GitHub Pages, Netlify).
2. Generate a QR code pointing to the deployed URL using a QR code generator.

## Files
- `index.html`: Main HTML structure
- `styles.css`: Stylesheet
- `script.js`: JavaScript for language switching
